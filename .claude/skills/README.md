# Claude Skills

Esta pasta contém skills customizadas para o Claude.

## Skills Disponíveis

| Skill | Descrição | Uso |
|-------|-----------|-----|
| `financial-analysis` | Análise de investimentos e empresas B3 | Relatórios, due diligence |
| `firecrawl` | Web scraping inteligente | Coleta de dados de sites |
| `automation` | Workflows e integrações | n8n, Make.com, APIs |
| `openclaw` | Assistente pessoal AI | Setup e configuração |
| `claude-mem` | Memória persistente entre sessões | Continuidade de contexto |

## Como Usar

As skills são ativadas automaticamente pelo Claude quando o contexto é relevante.

## Estrutura de uma Skill

Cada skill deve ter um arquivo `SKILL.md` com:
- Descrição do propósito
- Instruções de uso
- Exemplos práticos
- Referências e APIs
